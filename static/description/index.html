Gestión de Software - Empresa de Desarrollo

Módulo personalizado de Odoo 16 para gestionar una empresa de desarrollo de software.

Características Principales

Empresas Contratadoras

Gestión completa de los clientes que contratan proyectos a tu empresa.

Proyectos Extendidos

Extensión del módulo project de Odoo con campos adicionales como empresa contratadora, presupuesto y fecha de contratación.

Subtareas

Sistema de subtareas que los analistas pueden crear y asignar a los programadores.

Sistema de Roles

4 roles configurados: Administrador, Jefe de Proyectos, Analista y Programador, cada uno con permisos específicos.

Flujo de Trabajo

  1. Las empresas contratadoras encargan proyectos
  2. Los jefes de proyectos crean y organizan tareas
  3. Los analistas crean subtareas específicas
  4. Los programadores ejecutan las subtareas asignadas
